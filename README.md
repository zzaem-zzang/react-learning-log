# React Learning Log

`러닝 리액트(Learning React)`를 읽으며 예제 코드를 직접 작성하고 정리하는 학습 기록 저장소입니다.  
현재는 일부 챕터의 내용을 중심으로 정리하고 있지만, 여기에 멈추지 않고 React를 이해하는 데 필요한 개념들을 계속 확장하며 공부해 나갈 예정입니다.

## 프로젝트 소개

이 저장소는 책의 예제를 단순히 따라 작성하는 데서 끝내지 않고, 학습한 내용을 직접 실행 가능한 코드로 남기기 위해 만들었습니다.  
챕터별로 예제를 나누어 정리하면서 문법과 동작 원리를 확인하고, 이후에는 React 컴포넌트, 상태 관리, Hooks 등 더 넓은 범위까지 이어서 학습 내용을 축적해 갈 계획입니다.

## 학습 방향

- 책의 예제 코드를 직접 작성하며 개념 이해
- React 학습에 필요한 JavaScript 문법 선행 학습
- 현재 작성한 챕터 외의 내용까지 지속적으로 확장
- 단순 실습 저장이 아니라, 학습 과정과 성장 흐름이 보이는 기록 형태로 정리



## 저장소 의미

이 저장소는 현재 작성된 예제만을 모아둔 공간이 아니라, React 학습 전반을 단계적으로 확장해 나가기 위한 학습 아카이브입니다.  
이미 정리한 챕터보다 더 많은 내용을 앞으로 계속 공부하고 기록하면서, 기초 문법부터 React 핵심 개념까지 연결되는 흐름을 만들어 가는 것이 목표입니다.

- 학습 내용 요약과 회고
